#Project Overview

This application is a team-based project and task management system designed to help teams
collaborate and track work efficiently. A project can only be created within a team, ensuring
that all tasks and activities are organized around team collaboration. Team members are invited
to join projects via email. Each invited member must confirm the invitation before gaining access
to the project. The two Api's for sign up and login is completed with there login pages 

Once a team is formed and members have joined, the application allows users to:

      - Create and manage projects within a team
      - Assign and track tasks
      - Monitor task progress across the entire project
      - Maintain clear visibility of team responsibilities and workflow
